# NAU-CCL/Epymorph-276

Add a new ADRIO that provides daily state-level influenza data sourced from the COVID-19 Reported Patient Impact and Hospital Capacity by State Timeseries (RAW) dataset. It should allow selecting either hospitalization or admission counts via a configurable column option, support time frame and state scope context, and include missing-value checks even if the source data appears complete. Provide user-facing configuration consistent with existing ADRIO usage patterns.
